# **App Name**: Phần Mềm Vàng

## Core Features:

- Software Catalog: Display a catalog of software with Vietnamese content.
- Product Detail Pages: Showcase each software with descriptions, screenshots, and pricing.
- Category Browsing: Enable users to browse software by categories and subcategories, as found on the provided links.
- Search Functionality: Allow users to easily search for software using keywords.
- Purchase Links: Provide links to purchase software.
- AI-Powered Summaries: Offer a tool to generate summaries or translations of software descriptions using generative AI.

## Style Guidelines:

- Primary color: A rich gold (#D4AF37) to convey value and sophistication, suitable for software sales.
- Background color: A desaturated, dark grey (#333333) to provide contrast and make the gold elements stand out in a dark theme.
- Accent color: A deep red (#A62A2A), slightly analogous to gold, to highlight important call-to-action buttons and draw user attention.
- Body and headline font: 'Inter', a sans-serif font providing a modern and neutral appearance, suitable for both headlines and body text.
- Use clean and modern icons that visually represent software categories and features.
- Employ a clean and structured layout to present software information effectively. Prioritize readability and visual appeal.
